# AutoPro Platform - Project Snapshot (2026-03-07)

This is a comprehensive summary of where we are with the AutoPro project, prepared specifically for Google AI Studio.

## Overview
AutoPro is an imported car auction platform (Copart/IAAI style) serving the Libyan and Arab markets.

### Technology Stack:
- Frontend: React + TypeScript + Vite + TailwindCSS
- Backend: Express.js + Socket.IO + better-sqlite3
- Database: SQLite (auction.db)

    1. Purchase Invoice (Car + Commission) - Active immediately.

## Database Tables
- users: KYC data, balance, Buying Power (Deposit x 10).
- cars: Specs, damage info, live/upcoming/closed/offer_market statuses.
- bids: History of manual and auto-bids.
- transactions: Deposits, purchases, commissions.
- invoices: Multi-stage payments.
- notifications & messages: Internal communication system.
- shipments: Vertical timeline tracking for won vehicles.
- branch_configs: Multi-country settings (colors, currencies).

## Key Files
- server.ts: Central logic and DB initialization (~2.1k lines).

## How to Run
1. Run `npm install`
2. Run `npm run dev`
3. Visit `http://localhost:5173/api/debug/seed-simulation` to populate test data.
