## 📋 نظرة عامة (Overview)

AutoPro هي منصة متكاملة لمزادات السيارات المستوردة، تدعم تعدد الفروع (ليبيا، مصر، الإمارات، السعودية).

### 🛠️ التقنيات المستخدمة

- **Frontend**: React, TypeScript, Vite, TailwindCSS, Lucide Icons, Framer Motion.
- **Backend**: Express.js, Socket.IO (للتحديثات الحية), Better-SQLite3.
- **Database**: SQLite (`auction.db`).

## 🚀 آخر الإنجازات (Latest Progress)

### 1. إصلاح نظام الصور (Image Seeding Fix)

- تم تحديث روابط Unsplash في `server.ts` لتؤدي إلى صور سيارات حقيقية وبجودة عالية.
- تم إضافة نظام "باقة صور" للتنويع بين الصور الخارجية والداخلية لكل سيارة.

### 2. محاكي البيانات (Simulation Seeding)

- إضافة Endpoint: `GET /api/debug/seed-simulation`.
- يقوم هذا الرابط بإنشاء بيئة تجريبية كاملة:
  - 5 مشترين وهميين مع أرصدة مختلفة.
  - 20 سيارة موزعة على حالات (Live, Upcoming, Offer Market).
  - عمليات فوز سابقة، فواتير، رسائل داخلية، وإشعارات.

### 3. نظام "سوق العروض" (Offer Market)

- إذا انتهى المزاد ولم يصل السعر للحد الأدنى (Reserve Price)، تنتقل السيارة تلقائياً إلى "سوق العروض" لمدة يومين.
- يمكن للمشترين تقديم عروض، وللمدير قبول أو رفض العرض.

### 4. نظام الفواتير الثلاثي (Triple Invoice System)

- عند فوز المشترك بسيارة، يتم إنشاء 3 فواتير تلقائية:
    1. **فاتورة الشراء** (قيمة السيارة + عمولة).
    2. **فاتورة النقل الداخلي** (تفعل بعد دفع فاتورة الشراء).
    3. **فاتورة الشحن الدولي** (تفعل بعد وصول السيارة للمستودع).

### 5. مزامنة لوحات التحكم

- إضافة أنظمة الإشعارات (Notifications) والرسائل الداخلية (Messages) إلى لوحة المدير والمشتري.
- توحيد الهوية البصرية (Header/Sidebar) في جميع اللوحات.

## 📊 هيكل قاعدة البيانات (Database Schema)

- **users**: حسابات المستخدمين، الأدوار، الرصيد، القوة الشرائية، بيانات KYC.
- **cars**: بيانات السيارات، الأسعار، الحالات (live/upcoming/offer_market/closed)، الصور.
- **bids**: سجل المزايدات (يدوية وتلقائية/Proxy).
- **transactions**: العمليات المالية (إيداع، شراء، عمولات).
- **invoices**: الفواتير وحالات الدفع.
- **messages & notifications**: نظام التواصل الداخلي.
- **shipments**: تتبع الشحنات وحالاتها.
- **branch_configs**: إعدادات الفروع (الألوان، الشعارات، العملات).

## 📂 أهم الملفات (Key Files)

1. `server.ts`: المحرك الرئيسي (Backend + Socket.io + DB Initialization).
2. `src/pages/AdminDashboard.tsx`: مركز الإدارة والتحكم الشامل.
3. `src/pages/UserDashboard.tsx`: لوحة تحكم المشتري والمحفظة المالية.
4. `src/pages/SellerDashboard.tsx`: لوحة تحكم التجار ورفع السيارات.
5. `src/context/StoreContext.tsx`: إدارة الحالة العامة والتواصل مع الـ API.

## 🛠️ كيفية التشغيل (How to Run)

1. افتح Terminal في مجلد `autopro`.
2. شغّل `npm install` (إذا لم تكن مثبتة).
3. شغّل `npm run dev` لتشغيل السيرفر والواجهة معاً.
4. للبيانات التجريبية، افتح المتصفح على: `http://localhost:5173/api/debug/seed-simulation`.
